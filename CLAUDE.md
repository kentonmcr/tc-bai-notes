# Notes App

## What this app is

A minimal Apple Notes / Google Keep clone. Users create short notes with a
title and body, edit them, and delete them.

**How to run:** open `index.html` in a browser.

### Core interactions

- Click a "New note" button to create a note
- Type a title and body; changes save automatically or on a Save click
- Open a list of notes; click one to view or edit it
- Delete a note from the list
- Notes persist across reloads

## Stack

- HTML
- CSS
- JavaScript
- GitHub Pages (hosting)
- localStorage (persistence)

## Rules

- Do not add any libraries without asking first
- Keep everything in three files:
  1. `index.html`
  2. `style.css`
  3. `app.js`
